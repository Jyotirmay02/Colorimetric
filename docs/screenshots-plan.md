# Screenshots Plan — Play Store

Google Play requires **at least 2 phone screenshots**, but 6–8 looks far more credible. Capture on an actual Android device or a recent Pixel emulator in **portrait** at the standard 1080×1920 (or larger). Use a real solution photo for screen 4 to maximise authenticity.

Each row below is a separate screenshot. Caption text is optional in the actual screenshot but should match what's on the screen so reviewers grasp the value quickly.

| # | Screen to capture | Suggested caption / context |
|---|---|---|
| 1 | **Home** — top hero + 4 stat tiles + Quick Actions grid | "Build calibration curves. Predict concentration by colour." |
| 2 | **Calibrate tab** — sample list with 4–5 samples added, one tagged as Blank | "Add samples with known concentration. Tag blanks for IUPAC LoD." |
| 3 | **Analyze ROI** — image of a coloured vial with the ROI box overlaid and the extracted R/G/B + hex shown | "Tap any pixel — extract R, G, B in one tap." |
| 4 | **Analysis tab → Calibrate Analysis** — selected fit card with R² · SE · LoD stats and the scatter chart with best-fit line visible | "50+ equations, automatically ranked by R²." |
| 5 | **Analysis tab → Multi-equation overlay** — chart showing 4–6 overlaid fits with legend | "Compare up to 6 best fits side by side." |
| 6 | **Predict tab** — saved prediction card with the predicted µM value and the saved best equation label | "Predict unknowns. Every equation, ranked by trust." |
| 7 | **Filter / equations modal** — the equation picker modal open showing the list of linear equations with checkboxes | "Pick which equations to compare, including Beer-Lambert." |
| 8 | **Camera capture** — in-app camera UI with framing reticle and the locked-parameter badges (FLASH OFF · AF ON · Q · 1.0) | "Locked capture settings for cross-device consistency." |

## How to capture

- **Android phone**: hold Power + Volume Down → screenshots save to gallery → upload to Play Console
- **Emulator (Android Studio)**: use the side panel camera icon to save PNG
- **iPhone** (if testing iOS later): Power + Volume Up

## Tablet screenshots

7" tablet (1200×1920) and 10" tablet (1920×2560) screenshots are **optional** but help the listing. Skip for first launch — you can add them later.

## Tips

- Use real data (5–8 samples with realistic concentrations like 10, 25, 50, 75, 100 µM) — looks far more convincing than placeholder text.
- Pick samples that produce a clean R² (>0.95) for screen 4 — researchers spot bad fits instantly.
- For screen 3, photograph an actual coloured solution (e.g., copper sulfate, food colouring in water, methylene blue) against a plain white background.
- Make sure the navigation bar at the bottom shows the correct tab highlighted in each screenshot — adds polish.
